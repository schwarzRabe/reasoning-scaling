# reasoning-scaling

Сравнительный анализ методов улучшения рассуждений
компактных языковых моделей. Магистерская ВКР, МИФИ, 2026.

## Архитектура

Три слоя, зависимости только вниз:

    Слой 0  parsing, flops             чистые функции, ни от кого не зависят
    Слой 1  runner                     управление прогонами
    Слой 2  sft, grpo, inference,      каждый делает одно действие,
            evaluate, analysis         не знает о соседях

Ноутбуки (слой 3) собирают модули в пайплайн.
Весь pipeline виден в одном файле.

## Запуск

    00_setup       — окружение, модели, тесты
    01_data_prep   — данные, baseline
    02_sft         — 36 SFT-прогонов
    03_grpo        — выбор лучших, GRPO
    04_inference   — генерация, стратегии
    05_analysis    — Парето-фронт, графики

Последовательно, с ручным контролем между этапами.
Текст ВКР параллельно, финализация к 07.06.2026.

## Файлы

    rscaling/       библиотека (8 модулей)
    configs/        параметры (4 YAML)
    notebooks/      точки входа (6 ноутбуков)
    results/        метрики и статусы (git)
    data/           датасеты (.gitignore)

    docs/           отчёты и текст ВКР
